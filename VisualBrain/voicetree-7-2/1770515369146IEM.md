## 🔥 Critical Issues

### 🚨 BLOCKING BUG
**Claim/Unclaim Completely Broken** - Users see success but requests stay "Open"
- Root cause: RLS policy blocks UPDATE when `claimed_by IS NULL`
- Impact: Core feature unusable
- See: [[naars-cars-rides-and-favors-module.md|Rides & Favors Module]]
- See: [[database-schema-and-migrations.md|Database Schema]]
- See: [[architecture-overview-diagram.md|RLS Bug Visualization]]

### ⚠️ Performance Issues
- Message sorting on main thread (large conversations lag)
- Badge count queries scale poorly
- Cost estimation race condition

**Full Analysis:** [[naars-cars-project-summary-and-technical-debt.md|Project Summary & Technical Debt]]
